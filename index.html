Welcome to the blog!

Thursday 12-06-2014

This is the first iteration of my custom blog "framework". Head over to GitHub if you're interested in seeing how it works (in progress...).

Starting a blog isn't easy. Sure, there are plenty of frameworks out there, but it's really quite a daunting task to set one up. If you choose a framework, you have to go through the motions of figuring out where your vision fits into it. Besides being frustrating, it can be constraining. This, my latest attempt to start a blog, started with Ruby-on-rails (overkill), moved to Django (looks great, but need time to learn it), forayed into Hexo (too simple), and finally led to my favorite option: from-scratch.

I ended up doing what I always love to do: working from scratch. I set up a quick html page with content, got the css just right then introduced/explored jquery and WebGL. I set out to have this blog be my way of keeping up with the latest web technologies but I may actually decide to develop this further into a web framework (hell, everyone has one these days). Chances are I'm going to migrate this to Django soon, but in the meantime this is too much fun.

It's just the beginning though. My immediate next steps are to:

  • Figure out how to make this post text look nicer
  • Work more on generative methods; Posts should be separate entities/files separate from the formating. I might go with XML but it seems the XML/XSLT paradigm is dying
  • Start churning out content!

Stay tuned for more content (in both the Blog and other corners of my site). Cheers! -V
